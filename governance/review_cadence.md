# Review Cadence

| Artifact Type | Review Cycle | Trigger |
|--------------|-------------|---------|
| SSS initiatives | Quarterly | Calendar |
| SSS projects | Monthly | Calendar |
| SSS specifications | Quarterly | Calendar or change |
| Decision records | Annually | Calendar |
| Knowledge entries | Semi-annually | Calendar |
| Intelligence briefs | Before use | On demand |
| Governance artifacts | Annually | Calendar |
| Automation scripts | Quarterly | Calendar or change |
| Templates | Annually | Calendar |
| Ontologies | Quarterly | Calendar |

## Overdue Handling

1. Automation flags overdue artifacts
2. Owner notified
3. Owner reviews and updates, deprecates, or archives
