# Approval Workflow

## Levels

| Level | Authority | Applies To |
|-------|-----------|------------|
| L1 — Automated | CI/CD | All submissions |
| L2 — Peer | Any contributor | Knowledge entries, source notes |
| L3 — Expert | Domain expert | Specifications, technical content |
| L4 — Leadership | SSS leadership | Initiatives, strategic decisions |
| L5 — Legal | Legal counsel | SSS-facing documents |

## Requirements by Type

| Artifact Type | Minimum Level |
|--------------|--------------|
| Knowledge entry | L2 |
| Source note | L1 |
| Intelligence brief | L3 |
| Decision record | L4 |
| SSS initiative | L4 |
| SSS project | L3 |
| SSS specification | L3 |
| Governance artifact | L4 |
