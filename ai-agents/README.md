# ai-agents/ — AI Agent Assets

This directory contains assets designed specifically for AI agent consumption: prompt libraries, task guides, context injection protocols, and research methodologies.

## Structure

| Directory | Contents |
|-----------|----------|
| `prompt_library/` | Reusable prompt templates for common tasks |
| `task_guides/` | Domain-specific task guides |
| `context_injection/` | Context files designed for AI context window loading |
| `research_methodologies/` | Standard research methodologies |

## Usage

AI agents should:
1. Load relevant context injection files before task execution
2. Follow task guides for domain-specific work
3. Use prompt library templates for consistent outputs
4. Apply research methodologies for validated findings
