# Investor Materials Production Guide

## Overview

Produce investor-facing materials for SSS and its portfolio entities.

## Pre-Production Checklist

- [ ] Load sss_domain_context.md
- [ ] Load legal_gating_context.md
- [ ] Confirm engagement status wording
- [ ] Identify target investor type (strategic, financial, DFI)

## Production Rules

1. **Figures**: All labeled "illustrative" unless sourced
2. **Engagement Status**: Use exact wording from strategic thesis
3. **MBS/Anchor Investor**: Private investor only
4. **No State Iconography**: Zero government symbols on investor-facing assets
5. **Legal Gating**: Apply all legal gating rules

## Output Formats

- One-pagers (2-page)
- Investor decks (12-18 slides)
- Financial summaries
- Due diligence materials

## Quality Standards

- All claims sourced and confidence-tagged
- Cross-references to validated knowledge entries
- Review by SSS leadership before distribution
