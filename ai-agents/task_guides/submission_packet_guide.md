# Submission Packet Production Guide

## Overview

Produce government-ready, institutionally credible, legally cautious, and development-finance-aligned materials for SSS Somaliland-facing submissions.

## Pre-Production Checklist

- [ ] Load legal_gating_context.md
- [ ] Load visual_system_context.md
- [ ] Load sss_domain_context.md
- [ ] Confirm engagement status wording
- [ ] Identify target audience

## Production Steps

1. **Draft Content**: Ministerial, DFI-ready, investor-readable language
2. **Apply Legal Gating**: All "proposed", hedged terms, no unauthorized claims
3. **Apply Visual System**: Phase 3 formatting, colors, seals
4. **Quality Control**: Run through 53-point checklist
5. **Review**: Submit for expert review

## Output

- 16-page memorandum + 5 annexes (7 deliverable files)
- A4 portrait, deep near-black cover, warm cream interior
- All claims sourced and confidence-tagged
