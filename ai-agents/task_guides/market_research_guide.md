# Market Research Guide

## Overview

Standard process for conducting and documenting market research for SSS.

## Process

### 1. Scope Definition

- Define research question
- Identify relevant markets/geographies
- Set timeline and budget

### 2. Source Identification

Prioritize sources by evidence tier:
- T1: Government/official data
- T2: Academic/peer-reviewed
- T3: Industry/professional reports
- T4: Press/media

### 3. Data Collection

- Collect from multiple independent sources
- Document all sources with URLs and access dates
- Note any paywalled or restricted sources

### 4. Validation

- Cross-reference claims across sources
- Flag conflicts and discrepancies
- Assign confidence levels

### 5. Documentation

- Use knowledge_capture_template.md
- Include all sources with evidence tiers
- Tag confidence levels
- Cross-reference related artifacts

## Output

- Knowledge entry in knowledge/validated/
- Intelligence brief if synthesis required
- Gap report if information is missing
