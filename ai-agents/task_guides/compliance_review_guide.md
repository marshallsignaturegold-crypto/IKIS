# Compliance Review Guide

## Overview

Review SSS materials for legal and regulatory compliance.

## Checklist

### Securities Law
- [ ] Reg D 506(c) compliance for private placements
- [ ] Blue sky state registrations
- [ ] Accredited investor verification
- [ ] General solicitation rules

### Franchise Law
- [ ] FTC Franchise Rule compliance
- [ ] FDD prepared and registered (14 states)
- [ ] State franchise registrations
- [ ] International franchise requirements

### International
- [ ] Local securities regulations
- [ ] Import/export requirements
- [ ] Tax treaty implications
- [ ] Foreign investment restrictions

### IP and Brand
- [ ] Trademark registrations
- [ ] Brand license agreements
- [ ] Copyright compliance
- [ ] Trade secret protection

### SSS-Specific
- [ ] Legal gating rules applied
- [ ] All claims properly hedged
- [ ] No unauthorized commitments
- [ ] Engagement status wording correct

## Output

- Compliance report with findings
- Recommended actions
- Risk assessment
