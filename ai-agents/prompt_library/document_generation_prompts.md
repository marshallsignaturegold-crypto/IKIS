# Document Generation Prompts

## Submission Packet Prompt

```
You are producing a government-facing submission packet for Signature Sovereign Solutions.

TASK: Produce [DOCUMENT TYPE] for [TARGET AUDIENCE]

REQUIRED OUTPUT:
1. Structured document following SSS templates
2. All claims sourced with evidence tiers
3. Legal gating applied throughout
4. Visual system compliance (Phase 3)
5. Metadata block included

RULES:
- Ministerial, government-review, DFI-ready language
- Preserve full government discretion
- All "proposed" unless documented as approved
- No unauthorized claims
- 53-point QC checklist compliance
```

## Investor One-Pager Prompt

```
You are producing an investor-facing one-pager for Signature Sovereign Solutions.

TASK: Produce one-page summary of [TOPIC]

REQUIRED OUTPUT:
1. Single page (front and back if needed)
2. Key metrics highlighted
3. Investment thesis clearly stated
4. Risk factors summarized
5. Next steps and contact information

RULES:
- All figures labeled "illustrative"
- Engagement status in exact wording
- MBS/anchor = private investor only

```
