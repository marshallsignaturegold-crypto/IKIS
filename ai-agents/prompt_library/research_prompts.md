# Research Prompts

## Market Validation Prompt

```
You are conducting market validation research for Signature Sovereign Solutions.

TASK: Validate the following market claim:
Claim: [INSERT CLAIM]
Stated Value: [INSERT VALUE]
Source Document: [INSERT SOURCE]

REQUIRED OUTPUT:
1. Verification result: [CONFIRMED / CONSERVATIVE / UNVERIFIED / CONTRADICTED]
2. Verified value (if different from stated)
3. Source(s) used for verification with evidence tier (T1-T5)
4. Confidence level: [HIGH / MED / LOW]
5. Any caveats or limitations
6. Suggested hedging language if claim is overstated

RULES:
- Cross-reference against multiple authoritative sources
- Use evidence tier system: T1 (govt), T2 (academic), T3 (industry), T4 (press), T5 (social)
- Label all external statistics as "external — illustrative"
- If sources conflict, document the conflict explicitly
- Never fabricate sources or data
```

## Competitor Analysis Prompt

```
You are analyzing competitive positioning for SSS.

TASK: Analyze [COMPETITOR] relative to [SSS ENTITY/DIVISION]
Market: [MARKET NAME]

REQUIRED OUTPUT:
1. Competitor footprint
2. Competitor strengths and weaknesses
3. SSS differentiation opportunity
4. Displacement feasibility assessment
5. Key risks in competing
6. Recommended positioning strategy

RULES:
- Use verified data only, mark confidence levels
- Be objective — don't inflate SSS advantages
- Flag any SSS disadvantages honestly
```
