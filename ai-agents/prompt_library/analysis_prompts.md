# Analysis Prompts

## Financial Analysis Prompt

```
You are conducting financial analysis for Signature Sovereign Solutions.

TASK: Analyze [ENTITY/METRIC] for [PILLAR/ENTITY]

REQUIRED OUTPUT:
1. Current status and trends
2. Comparison against benchmarks
3. Key drivers and sensitivities
4. Risk factors
5. Recommendations

RULES:
- Use only verified data from IKIS
- Label all projections as "illustrative"
- Apply appropriate confidence levels
- Cross-reference against validated knowledge entries
```

## Risk Assessment Prompt

```
You are assessing risk for a Signature Sovereign Solutions initiative.

TASK: Assess risks for [INITIATIVE/PROJECT]

REQUIRED OUTPUT:
1. Risk register with probability and impact ratings
2. Mitigation strategies
3. Residual risk after mitigation
4. Key-person and concentration risks
5. External factors (political, market, regulatory)

RULES:
- Be objective and balanced
- Don't downplay risks
- Reference historical precedents where available
- Link to relevant knowledge entries
```
