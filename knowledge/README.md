# knowledge/ — General Knowledge Artifacts

This directory contains general knowledge artifacts not specific to SSS business domain.

| Directory | Content Type | Status |
|-----------|-------------|--------|
| `validated/` | Validated knowledge entries | Curated, reviewed |
| `source_notes/` | Raw source notes | Unprocessed |
| `derived/` | Derived intelligence | Synthesized |

## Usage

- Add validated knowledge to `validated/`
- Add raw research to `source_notes/`
- Add intelligence briefs and gap reports to `derived/`
- Follow templates and metadata specification
